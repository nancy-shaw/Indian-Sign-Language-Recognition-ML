# Literature Review

This directory contains summaries and analysis of research papers studied during the development of the Indian Sign Language (ISL) Recognition project.

The objective of this review is to:

* Understand existing approaches for sign language recognition.
* Analyze datasets used in previous research.
* Compare machine learning and deep learning models.
* Identify limitations in current systems.
* Explore opportunities for improving real-time ISL recognition.

Each paper summary includes:

* Research objective
* Methodology
* Dataset information
* Models used
* Results and performance
* Key learnings relevant to our project
