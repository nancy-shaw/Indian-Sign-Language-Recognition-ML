# Phase 01: Literature Review

## Status

Completed

---

## Objective

To understand the current state of research in sign language recognition and identify suitable approaches for the Indian Sign Language (ISL) Recognition project.

---

## Activities Performed

* Studied multiple research papers related to sign language recognition.
* Explored traditional machine learning approaches.
* Investigated deep learning architectures such as CNNs, LSTMs, and GRUs.
* Analyzed MediaPipe-based recognition systems.
* Examined real-time sign language recognition pipelines.
* Reviewed publicly available datasets and benchmark studies.

---

## Key Learnings

* Data preprocessing plays a critical role in recognition accuracy.
* MediaPipe provides an efficient solution for hand landmark extraction.
* Deep learning models outperform traditional methods for complex recognition tasks.
* Sequence models such as LSTM and GRU are useful for dynamic gesture recognition.
* Dataset quality significantly impacts model performance.

---

## Outcome

The literature review phase helped establish the project's technical direction and provided a strong foundation for dataset preparation and model development.
